
SkillStamp is a web-based platform that enables professionals to showcase verified skills and certificates, while helping recruiters find qualified candidates. With secure authentication, certificate management, and intelligent job matching, SkillStamp streamlines the hiring process for both job seekers and employers.

## ✨ Key Features

- Secure authentication with role-based access (Job Seeker & Recruiter)
- Create and manage professional skill profiles
- Upload and verify certificates
- Post and manage job openings
- Apply for jobs and track application status
- Skill-based candidate and job matching
- Recruiter dashboard for managing applicants
- Responsive and user-friendly interface

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Python, Flask
- **Database:** MySQL

## ⚙️ Workflow

- User/Recruiter Registration & Login
- Skill and Certificate Management
- Secure Certificate Upload
- Job Posting and Applications
- Application Tracking
- Flask Backend with MySQL Database
